# TP4_Martinez_Fabris

Request-Reply with ZeroMQ:
- Server in Python (REP)
- Client in Java (REQ)

## Python server

```bash
source .venv/bin/activate 
python3 -m pip install -r requirements.txt
python3 server.py
```

## Java client

```bash
mvn compile
mvn exec:java
```
